# DrCalorieAI-
Dr Calorie AI: Personalized Calorie Tracker

The manual workflow requires two separate terminals:
This web uses Ollama model llama3.2 

1.**Terminal 1:** Boot up ollama server:
```bash powershell
set OLLAMA_ORIGINS="*"; ollama serve
```

2. **Terminal 2:** Run the Python server directly:
```bash
py server.py
    *(Make sure you are inside the `backend` folder first!)*
```
